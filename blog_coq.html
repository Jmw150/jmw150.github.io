Coq and Javascript

To start, JSCoq may have been written, and now maintained, poorly. No other verification software seems to be going for webstuff either. And this may be why it is uncommon to see verification software on the internet.

JSCoq more less should just work, but does not. The standard Coq compiler outputs json code. There is some irony that some programmers of sophisticated type systems cannot create functional internet code. Hopefully someone with more competence will do a proper portable translation someday.

Coqdoc, on the other hand, is a pretty reasonable compiler that outputs static html from Coq code. So there is no shortage of functional tutorials of Coq online. That is at least good.
